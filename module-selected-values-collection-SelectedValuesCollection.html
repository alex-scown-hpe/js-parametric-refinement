Class: SelectedValuesCollection

selected-values-collection~ SelectedValuesCollection

new SelectedValuesCollection()

Collection designed to be the only mutable store for the selected parametric values in an application. Every model
should have string "field" and "value" attributes.

The Backbone.Collection.modelId method is used to ensure that models with different field and value are considered
unique, without having to specify an id on each model explicitly.
